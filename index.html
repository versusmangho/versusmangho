사용 가이드

1. 플레이어 & 매칭

  • 등록: 닉네임을 입력하고 엔터를 누르세요. (최대 8명)
  • 매칭: 대기 목록에서 두 명의 플레이어를 클릭하면 매칭이 성사됩니다. 먼저 클릭한 사람이 선택자(원디골)가 됩니다.
  • 우선순위 (가중치): 기본적으로 가중치 점수가 높은 순으로 정렬됩니다. 가중치는 현재 대기, 평균 대기, 원디골 비율 등을 종합하여 계산된 값입니다.
  • 테이블 정렬: '가중치', '닉네임' 등 테이블의 제목(헤더)을 클릭하면 해당 기준으로 목록을 오름차순/내림차순 정렬할 수 있습니다.
  • 보류/복귀: 플레이어 목록의 '보류' 버튼을 눌러 잠시 매칭에서 제외할 수 있습니다. '복귀'를 누르면 보류한 시간만큼 대기 시간이 보정되어 복귀됩니다.
  • 신입 우선권: 이 옵션을 켜면, 방에 새로 들어온 플레이어('신입' 태그)가 빠르게 게임을 할 수 있도록 가중치에 보너스를 받습니다.
  • 세이프가드: 이 옵션을 켜면, 4판 이상 대기한 플레이어가 있을 경우 다른 모든 규칙을 무시하고 해당 플레이어에게 최상위 우선순위가 부여됩니다.

2. 📸 입장/퇴장 분석기

  • 사용법: 화면 아무 곳에나 버망호 로비의 스크린샷을 Ctrl+V로 붙여넣으세요.
  • 작동 원리: 두 장의 스크린샷(이전/현재)을 비교하여 자동으로 분석합니다.
  • * 이미지 유사도를 기반으로 판단하기에, 가끔 오류가 날 수 있습니다. (특히 레디<->레디해제 중 스크린샷이 찍히는 경우)
  • 분석 결과:
    • 🔴 퇴장: 방을 나간 인원
    • 🔵 잔류: 자리를 지킨 인원
    • 🟢 입장: 새로 들어온 인원

3. 기타

  • 자동 저장: 모든 데이터는 브라우저에 자동 저장되어 새로고침해도 유지됩니다.
  • 실행 취소/다시 실행: 실수로 매칭을 하거나 플레이어를 잘못 삭제했을 때 이전/다음 상태로 되돌릴 수 있습니다.
  • 1순위 문구 복사: '현재 1순위 문구 복사' 버튼을 누르면, 현재 가장 우선순위가 높은 플레이어의 대기 상태를 설명하는 문구가 클립보드에 복사됩니다.
  • 초기화: '전체 초기화' 버튼을 누르면 모든 데이터가 삭제됩니다.

티어별 난이도

티어를 선택한 뒤, 복사 버튼을 누르면 채팅에 붙여넣기 좋게 티어별 난이도가 클립보드에 복사됩니다.
